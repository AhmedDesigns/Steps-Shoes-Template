# Steps-Shoes Template

A modern, responsive e-commerce website for branded sneakers including **Nike, Adidas, Puma, New Balance**, and more. Designed for desktop and mobile, featuring a clean UI and smooth user experience.

## Features

- **Hero Banner with Background Video:** Engaging video autoplay and loop for maximum impact.
- **Category Sections:** Men, Women, and Kids with attractive images.
- **Responsive Design:** Works seamlessly on desktop, tablet, and mobile devices.
- **Modern UI/UX:** Clean typography, clear call-to-action buttons, and intuitive layout.
- **Easy to Customize:** Images, videos, text, and styles can be easily replaced.
